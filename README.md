# U-Trailers

A React JS Project with Suspense to show latest trailers of upcoming movies.

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```
### Project Stack:
 * React JS
 * Suspense
 * Lazy
 * Bulma CSS Framework
